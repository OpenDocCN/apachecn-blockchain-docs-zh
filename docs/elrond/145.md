 erdpy 的命令行界面

**erdpy** 公开了一些**命令**，组织在**组**中。这里的[列出了完整的命令行界面](https://github.com/ElrondNetwork/elrond-sdk-erdpy/blob/master/erdpy/CLI.md)。命令用法和描述可通过`--help`标志获得。

例如:

```
erdpy --help
erdpy tx --help
erdpy tx new --help 
```
