# Eth 2.0 信标 API

> 原文:[https://web3py.readthedocs.io/en/stable/web3.beacon.html](https://web3py.readthedocs.io/en/stable/web3.beacon.html)

警告

这个 API 是实验性的。客户端支持是不完整的，API 本身仍在发展。

要使用这个 API，您需要一个本地或远程运行的信标节点。要进行设置，请参考您的特定客户端的文档。

有了正在运行的信标节点后，导入并配置信标实例:

```
>>> from web3.beacon import Beacon
>>> beacon = Beacon("http://localhost:5051") 
```
